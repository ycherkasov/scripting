• Note: Set Option key in iTerm Preferences > Profiles > Keys menu. Usually it's Esc or Alt key
  • General commands
    • Cmd+Enter Fullscreen On/Off
    • Cmd+F Search with Regexp support
    • Cmd+; Command history dropdown menu
    • Cmd+Shift+H Paste history
  • Tabs
    • Cmd+T New Tab
    • Cmd+W Close Tab or Window
    • Cmd+Number Key Go to Tab
    • Cmd+Option+Arrow Key Go to Next Right/Left Tab
  • Panes
    • Cmd+D Split Window Vertically (same profile)
    • Cmd+Shift+D Split Window Horizontally (same profile)
    • Cmd+Option+Arrow Key Go to Right/Left Pane
    • Ctrl + D Close Current Pane
    • Cmd+Shift+Enter Maximize Pane
    • Cmd+Ctrl+Arrow Resize Pane
    • Cmd+], Cmd+[ Go to Next Right/Left Pane
    • Cmd+Alt+Shift Move a pane with the mouse (press and drag and drop the pane)
    • Cmd+Alt+I Broadcast command to all panes in window
  • Cursor navigation
    • Cmd+Arrow Keys Move back and forth on a line by words using Arrows
    • Option+F/B Move back and forth on a line by words using Option
    • Ctrl+A or Home Move to the start of line
    • Ctrl+E or End Move to the end of line
    • Ctrl+U Delete to start of line
    • Ctrl+K Delete to end of line
    • Up Arrow Repeat last command
    • Ctrl+W Delete previous word
    • Ctrl+L or Cmd+K Clear screen
  • Search the Command History
    • Ctrl+R Search as you type; Repeat Ctrl + R to loop through result
    • Ctrl+R twice Search the last remembered search term
    • Ctrl+Y End the search at current history entry
    • Ctrl+G Cancel the search and restore original line
  • Mouse and trackpad usage
    • By default the middle mouse button performs "paste"
    • Double-clicking selects a whole word
    • Triple-clicking selects an entire line
    • Quadruple-clicking or three-finger tap performs a "smart select", matching recognized strings such as URLs and email addresses
    • If you hold Cmd+Alt while selecting, a rectangular selection will be made (cursor in this mode has a cross shape)
    • Right-clicking on a number shows its conversion to or from HEX
    • Right-clicking on a number which looks like a Unix timestamp shows its representation in local time
    • Right-clicking on a non-ASCII character shows its UTF-8 representation
