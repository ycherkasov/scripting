• Note: Set Option key in iTerm Preferences > Profiles > Keys menu. Usually it's Esc or Alt key
  • General commands
    • Ctrl+R Rescan active panel
    • Ctrl+X Q Quick View Mode in opposite panel
    • Ctrl+X I Info Mode in opposite panel
    • Ctrl+X D Compare directories
    • Meta+I Panel Encoding Menu
    • Ctrl+U Swap panels
    • Meta+C Quick cd
    • Meta+? Find File dialog
    • Meta+H Command history
    • Ctrl+\ Directory shortcuts list
    • Meta+G Move the selection to the first visible entry
    • Meta+R Move the selection to the middle visible entry
    • Meta+J Move the selection to the last visible entry
    • Meta+S Quick filename search
    • Meta++ Select by mask
    • Meta+- Remove selection
    • Meta+* Invert selection
    • Meta+I Copy to opposite panel
  • File and directory actions
    • Ctrl+X C Chmod for current selection
    • Ctrl+X ) Chown for current selection
    • Ctrl+X L Hardlink for current selection
    • Ctrl+X L Symlink for current selection
    • Ctrl+X Ctrl+D Compare files
    • Ctrl+Space Show directory size
    • Meta+I Open selected directory in opposite panel
    • Meta+Enter Copy selected filename to command line
